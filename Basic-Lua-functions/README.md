# Basic syslib functions

This folder contains separate Lua examples for common syslib object operations.

## Files
- [GetChildObject.lua](GetChildObject.lua) - Gets a child object by name.
- [GetChildren.lua](GetChildren.lua) - Lists all child objects.
- [GetCommunicationStatus.lua](GetCommunicationStatus.lua) - Checks communication status values.
- [CheckObjectDeleted.lua](CheckObjectDeleted.lua) - Checks whether an object is deleted.
- [GetModel.lua](GetModel.lua) - Returns the model type.
- [GetModifiedTime.lua](GetModifiedTime.lua) - Returns the last modified time.
- [GetParent.lua](GetParent.lua) - Returns the parent object.
- [GetPath.lua](GetPath.lua) - Returns the object path.
- [GetType.lua](GetType.lua) - Returns the object type and class code.
- [CheckWarningState.lua](CheckWarningState.lua) - Checks whether the object is in warning state.
- [DisableObject.lua](DisableObject.lua) - Disables an object using an object specification.
